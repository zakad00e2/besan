# Google Search Console Handoff

Complete these steps after the SEO changes are deployed to
`https://www.besankhalaily.com`.

1. Add or verify the `www.besankhalaily.com` domain property in Google Search
   Console.
2. Open **Sitemaps** and submit
   `https://www.besankhalaily.com/sitemap.xml`.
3. Inspect `https://www.besankhalaily.com/` and request indexing.
4. Confirm Google selects the inspected Arabic home page as its canonical.
5. Confirm the Arabic and English alternate URLs are discovered.
6. Test the Arabic home page in Google's Rich Results Test and confirm that the
   Person, ProfessionalService, and WebSite JSON-LD parses without errors.
7. Recheck indexing and search performance after Google recrawls the site.

Do not submit `/auth` or any `/dashboard` URL for indexing.
